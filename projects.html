  • CropCare-Smart Farmer Assistant

    I made this project after a 5-Day AI Agents: Intensive Vibe Coding Course With Google by using the learnings from the course. Using the Antigravity tool for the whole process.
    Developed a web application focused on crop management leveraging Vibe coding and Antigravity for efficient coding and production. Utilized GitHub for version control and code management. Deployed the application using Render for seamless access.
    1. Purpose & UX: CropCare is a premium glassmorphic web application (SPA) designed to empower Indian farmers with data-driven insights through an interactive, dark-themed dashboard.
    2. Key Benefits: Offers climate diagnostics, personalized crop suitability scoring, mandi price analysis (with 6-month history), NPK-to-organic calculations, and a local-storage-backed crop growth timeline.
    3. Frontend Stack: Built with semantic HTML5, vanilla CSS3 (utilizing custom properties, grid layouts, and backdrop filters), and modern ES6+ JavaScript (HTML5 Canvas API, LocalStorage).
    4. AI Integration: Powered by a custom conversational AI agent developed using the Google Agent Development Kit (ADK) and Gemini LLMs.
    5. Backend & Tooling: Supported by a Python-based Model Context Protocol (MCP) server, automated pytest suites, and local deployment via agents-cli.

  • Simon Game

    The Simon Game is a classic electronic memory game where players must recall and repeat an ever-increasing sequence of random colors and tones. Built using JavaScript, the web version typically features four colored panels (green, red, yellow, and blue) that light up and play a sound in a specific order. The script uses arrays to track both the game's generated pattern and the player's inputs, relying on event listeners for clicks and setTimeout functions to control the timing of the flashes. Every time the player successfully replicates the sequence, JavaScript adds a new step to the chain, pushing the player's pattern-recognition skills to the limit until a mistake triggers a "game over."

  • AuraGallery

    The Aura Gallery is a dynamic, responsive web application built using HTML, CSS, and vanilla JavaScript designed to showcase high-quality, real-time visual content. Created as a practice project to master asynchronous web development, the platform integrates seamlessly with the Unsplash API to fetch and render fresh, high-resolution imagery on demand. JavaScript handles the core data-fetching logic using fetch() and promises, parsing JSON data from the API to dynamically inject image elements, photographer credits, and download links directly into the DOM. The frontend features a clean, modern layout structured with semantic HTML and styled with advanced CSS features like Flexbox or Grid to create a responsive, fluid masonry layout that adapts perfectly to desktop and mobile screens alike. This project serves as an excellent portfolio piece demonstrating a solid grasp of API integration, handling HTTP requests, asynchronous state management, and modern UI/UX design principles.
